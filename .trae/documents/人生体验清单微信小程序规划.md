# 人生体验清单微信小程序规划

## 项目概述

创建一个"人生体验清单"微信小程序，小程序的名称为：打卡人生。MVP版本主打"学生时代"的人生体验，用户可以通过点击按钮选择自己完成过的体验，最后生成并分享个人专属的人生体验清单图片。

## 技术栈

* 微信小程序原生开发

* WXML + WXSS + JavaScript

* 云开发（可选，用于后续扩展）

## 项目结构


## 核心功能

### 1. 体验清单展示

* 固定生成50条学生时代相关的人生体验选项

* 每个选项以按钮形式呈现

* 采用响应式布局，适配不同屏幕尺寸

### 2. 交互功能

* 点击按钮实现选中/取消选中状态切换

* 选中状态按钮高亮显示

* 记录用户选中的体验项

### 3. 分享功能

* 底部吸底按钮"分享人生"

* 点击后生成包含用户选中体验的图片

* 将图片保存到手机本地

## 设计方案

### 1. 整体风格

* **模式**：Dark模式

* **主色调**：C4A2FF（品牌色和主按钮颜色）

* **辅助色**：

  * 选中状态：C4A2FF（高亮）

  * 未选中状态：333333（深色背景）

  * 文字颜色：FFFFFF（浅色文字）

### 2. 布局设计

* 顶部：页面标题"人生体验清单（学生时代）"

* 中部：体验选项列表（网格布局，每行2-3个按钮）

* 底部：吸底分享按钮

### 3. 组件设计

* **体验项按钮**：

  * 圆角矩形设计

  * 未选中时：深色背景，浅色文字

  * 选中时：C4A2FF背景，白色文字

  * 点击有轻微动画效果

* **分享按钮**：

  * 吸底固定位置

  * C4A2FF背景，白色文字

  * 宽度100%，高度适配

## 关键实现步骤

1. **初始化项目**：创建微信小程序项目，配置基本结构
2. **数据准备**：编写50条学生时代体验数据
3. **页面布局**：实现主页面布局和样式
4. **组件开发**：创建体验项组件
5. **交互逻辑**：实现按钮点击选中/取消选中功能
6. **图片生成**：实现选中体验的图片生成功能
7. **分享功能**：实现图片保存到本地功能
8. **测试优化**：测试各功能模块，优化用户体验

## 注意事项

1. **性能优化**：

   * 合理使用setData，避免频繁更新

   * 图片生成时注意内存管理

2. **用户体验**：

   * 按钮点击反馈清晰

   * 图片生成过程有加载提示

   * 保存图片后有成功提示

3. **兼容性**：

   * 适配不同微信版本

   * 适配不同屏幕尺寸

4. **权限处理**：

   * 申请相册权限用于保存图片

   * 处理权限拒绝的情况
